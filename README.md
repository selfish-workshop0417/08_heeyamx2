# team_calender

> 팀원들끼리 회사 일정 공유, 재택 여부 공유, 휴가 여부 등을 달력 형태로 확인하는 사내 툴

## 프로젝트 개요

- **서비스 유형**: 사내 툴
- **타겟 사용자**: 대행사 직원들
- **핵심 기능**: 팀별 캘린더 접속, 회사 일정 등록 및 재택 여부 표시, 휴가 예정/확정 관리

## 주요 특징

- 이메일+비밀번호 로그인
- 팀별 캘린더 분리 (전사 통합 뷰 없음)
- 휴가 확정 시 구글 캘린더 연동 + 팀 메일 자동 공유

## 폴더 구조

```
01-기획/       # 무엇을 만들 것인가
02-디자인/     # 어떻게 보여줄 것인가
03-CX/         # 누가 어떻게 사용할 것인가
04-마케팅/     # 어떻게 알릴 것인가
05-전략/       # 어떻게 지속 가능하게 만들 것인가
```
