# team_calender 설계 문서

**작성일**: 2026-04-17  
**프로젝트**: 대행사 사내 팀 캘린더 툴

## 1. 인증

- 누구나 이름 + 비밀번호로 회원가입
- 로그인 후 세션 유지 (재접속 시 자동 로그인)

---

## 2. 메인 화면 구조

```
[전체] [재택] [휴가] [회사일정]          ← 탭
┌─────────────────────┬────────────┐
│  < 2026년 4월 >     │ ☑ 팀원A   │
│                     │ ☑ 팀원B   │
│  달력 (월간 뷰)     │ ☐ 팀원C   │
│  오늘 날짜 하이라이트│            │
│  ← 이전달 / 다음달→ │            │
└─────────────────────┴────────────┘
```

- **상단 탭**: 전체 / 재택 / 휴가 / 회사일정
- **달력**: 월간 뷰, 이전·다음달 이동 가능 (과거·미래 모두)
- **오늘 날짜**: 하이라이트 표시
- **오른쪽 패널**: 팀원 체크박스 (다중 선택 필터)
- **회사 일정**: 여러 명 참여해도 달력에 1번만 표시, 참여자 이름 함께 노출

```
예시) [4월 18일] 워크숍 10:00~18:00
                 👥 팀원A, 팀원B, 팀원C
```

## 3. 일정 등록

### 등록 플로우
1. **구분 선택**: 휴가/언플 · 재택 · 일정
2. **시간 선택**: 재택 제외 모두 날짜+시간 입력 (재택은 날짜만)
3. **일정 이름 입력**

### 상태 플로우 (예정 → 확정)

- 휴가/언플 등록 시 이름 뒤에 자동으로 `(예정)` 추가
- 해당 일정 클릭 → **"확정"** 버튼 누르면 `(예정)` 제거

```
"여름휴가(예정)"  →  [확정]  →  "여름휴가"
"제주언플(예정)"  →  [확정]  →  "제주언플"
```

- 휴가 확정 시: 구글 캘린더 자동 등록 + 팀메일 자동 발송

### 회사 일정 참여
- 기존 회사 일정 클릭 → **"나도 참여"** 버튼으로 내 이름 추가 가능

### 수정/삭제
- 내가 등록한 일정은 수정·삭제 가능

---

## 4. 기술 요구사항

- **플랫폼**: 웹 브라우저 (PC 기준)
- **인증**: 이름 + 비밀번호, 세션 유지
- **외부 연동**: Google Calendar API + 팀 메일 자동 발송 (휴가 확정 시)
- **데이터 저장**: 일정별 이름, 날짜, 시간(재택 제외), 유형, 참여자, 상태(예정/확정)
