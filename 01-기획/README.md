# 01-기획

> 무엇을 만들 것인가 — 목적 정의, 기능 범위, 요구사항

## 담당 영역

기획팀은 PRD의 다음 섹션을 주도적으로 작성합니다:

- **1. 배경 및 목적** — 왜 이 서비스가 필요한가
- **2. 목표 및 성공 지표** — 무엇을 달성하면 성공인가
- **3. 타겟 사용자** — 누가 쓰는가
- **4. 핵심 기능 정의** — 무엇을 만드는가
- **5. 사용자 흐름 (User Flow)** — 어떻게 사용하는가

## 핵심 기능 요약

| 기능 | 설명 |
|------|------|
| 팀 캘린더 | 팀별 월간 캘린더, 타 팀 열람 불가 |
| 재택 표시 | 출근 / 재택 / 외근 상태 등록 |
| 회사 일정 | 관리자가 공통 일정(공휴일, 행사) 등록 |
| 휴가 신청 | 신청 → 승인 → 확정 플로우 |
| 구글 캘린더 연동 | 휴가 확정 시 자동 등록 + 팀 메일 발송 |

## 범위 외 (Out of Scope)

- 전사 통합 캘린더 뷰 (팀별 분리만 제공)
- 모바일 앱 (웹 브라우저만 지원)
- 타 팀 일정 열람
